Presentación del Proyecto

El Sistema de Gestión de Biblioteca es una aplicación web desarrollada para administrar de manera eficiente los procesos de registro, control y préstamo de libros mediante una base de datos relacional en SQL Server y la implementación de operaciones CRUD en ASP.NET.

El sistema garantiza un acceso rápido, confiable y seguro a la información bibliográfica, mejorando la experiencia de los usuarios y optimizando la gestión administrativa.

Estructura de la Base de Datos

Tablas Principales

  • Usuarios - Datos de lectores registrados
  • Autores - Información de escritores
  • Libros - Recursos centrales del sistema
  • Préstamos - Transacciones de entrega
  • Reservas - Solicitudes anticipadas

Relaciones

  • Usuarios ↔ Préstamos (1 a muchos)
  • Usuarios ↔ Reservas (1 a muchos)
  • Autores ↔ Libros (1 a muchos)
  • Libros ↔ Préstamos (1 a muchos)
  • Libros ↔ Reservas (1 a muchos)

Estructura de Tablas

Tabla Campos Principales
Usuarios UsuarioID, Nombre, Apellido, email, Teléfono, FechaRegistro
Autores AutorID, Nombre, Nacionalidad, FechaNacimiento
Libros LibroID, Título, AutorID, ISBN, AñoPublicación, Estado
Préstamos PréstamoID, UsuarioID, LibroID, FechaPréstamo, FechaDevolución, Estado
Reservas ReservaID, UsuarioID, LibroID, FechaReserva, Estado

Operaciones CRUD Implementadas

Crear (Create)

Registro de nuevos usuarios, libros, autores, préstamos y reservas.

Leer (Read)

Consulta de información existente en todas las tablas de la base de datos.

Actualizar (Update)

Modificación de registros existentes, como cambiar el estado de un libro o actualizar información de usuario.

Eliminar (Delete)

Eliminación de registros obsoletos o erróneos del sistema.
